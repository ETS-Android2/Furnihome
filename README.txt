In order to run our application you will need to install Google ARCore version
1.15 in your android device or emulator. The apk can be found in the link below:
https://www.apkmirror.com/apk/google-inc/arcore/arcore-1-15-2001210-release/
Our application requires a system image with the tag Google APIs (If you
run the application in android studio). In our project we used the Pixel 2 
device with API 29 (Google APIs).
